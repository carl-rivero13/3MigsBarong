# 3MigsBarong
E-Commerce Project about Barongs created using PHP Laravel 

## Access the File System using Local Host ##
1. git clone the repository
2. Download and Open the Xampp
3. Create a database named "your_local_database_name" in http://localhost/phpmyadmin
4. Open 3MigsBarong using LocalHost with http://localhost//3MigsBarongPHP/home.php
